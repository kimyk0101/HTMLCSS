inline-block의 특징

  • inline과 block의 특징을 모두 가짐
  • inline 요소들처럼 현재 라인에 이어서 표시됨
  • block 요소들처럼 width, height, margin, padding을 설정할 수 있음
  • 내부 콘텐츠의 크기에 따라 크기가 자동 조절됨. 블록 요소와 달리 전체 너비를 차지하지 않으며 필요한 만큼의 공간만 사용함
  • 수직 방향으로 정렬될 수 있음. inline-block요소가 한줄에 배치되면 vertical-align 속성을 사용하여 상대적으로 정렬할 수 있음
